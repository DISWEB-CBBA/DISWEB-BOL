Sistema Administrativo para Restaurante

Buenazo Almuerzos Saludables

Este sistema fue desarrollado en Microsoft Excel con macros (VBA) para gestionar de forma completa un restaurante de almuerzos saludables. Incluye inicio de sesión, menú principal, registro de compras, control de inventario, ventas, reportes, control de stock y mucho más. Está diseñado para mejorar la eficiencia del negocio y centralizar toda la operación del restaurante en una sola herramienta. Ideal para pequeños y medianos emprendimientos gastronómicos que deseen una solución sin depender de software costoso.
