Proyecto: Luz y Esperanza

Sistema web desarrollado en C# para la gestión integral de una farmacia. Permite administrar ventas, inventario, control de usuarios y operaciones por sucursal. Incluye interfaces intuitivas, reportes y módulos de seguridad. Diseñado para optimizar el flujo de trabajo y facilitar el control en tiempo real.
